Railway → projet openswissdata → service api → onglet Variables → + New Variable :

Pourquoi c'est non-négociable

Stripe LIVE est actif depuis le 25/04, mais aucun vrai paiement n'a jamais traversé le webhook avec la signature LIVE. Si une typo Railway env, un retry loop sur Resend, un volume R2 désynchronisé, ou une race condition existe → ton 1er client réel sera le cobaye. Mauvais. C'est toi qui dois être le cobaye d'abord.

Coût du test : ~1 200 CHF (achats sur ton propre compte) que tu refunderas immédiatement après. Stripe te facture juste les fees ~30 CHF (~2.5% × 1200) — c'est ton ticket d'assurance.
