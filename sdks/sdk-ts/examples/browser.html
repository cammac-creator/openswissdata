openswissdata browser SDK demo

Loads @openswissdata/sdk from a CDN as an ES module and runs a TARES lookup against the live MCP endpoint. 

Click "Lookup" to call https://mcp.openswissdata.com

Replace the import URL below with the published CDN URL once @openswissdata/sdk ships on npm. During local development use ../dist/esm/index.js after running npm run build.
