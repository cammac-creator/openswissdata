# openswissdata-samples

Example notebooks and integration patterns for [openswissdata.com](https://openswissdata.com) datasets.

## Notebooks

- [`tares_sap_integration.md`](./notebooks/tares_sap_integration.md) — Map Swiss HS8 codes into SAP GTS staging tables
- [`noga_nace_crosswalk.md`](./notebooks/noga_nace_crosswalk.md) — Translate Swiss company classifications between NOGA 2008, NOGA 2025, and NACE 2.1 for EU parent reporting
- [`finma_contrapartie_audit.md`](./notebooks/finma_contrapartie_audit.md) — Cross-check a list of Swiss counterparties against the FINMA authorised registry

Full `.ipynb` versions to be added once Alain runs them end-to-end with real datasets.
